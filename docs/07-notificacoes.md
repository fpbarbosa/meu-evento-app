# Notificações

Este documento descreve a arquitetura e o funcionamento do sistema de notificações do Meu Evento.

---

## Objetivo

Garantir comunicação em tempo real entre:

- Cliente
- Fornecedor
- Sistema

As notificações são fundamentais para melhorar experiência e engajamento.

---

# Serviço Utilizado

Firebase Cloud Messaging (FCM)

O FCM permite:

- Envio de notificações push
- Entrega para Android, iOS e Web
- Envio direcionado por dispositivo ou usuário
